## Interaction 53: Advanced UPI Spend Tracking UI

### User
agrawalpriya

### User Request
> The current method of adding a UPI split slider under every single expense category makes the UI cluttered. Can we move this functionality into an 'Advanced' sub-tab to make the primary expense entry cleaner? The advanced tab could perhaps control the split for entire categories at once.

### AI Response Summary
Of course. That's a great UX suggestion that simplifies the primary task of entering expenses while still providing powerful control over payment methods. I have refactored the "Monthly Spending" page to use a new tabbed interface.

1.  **New Tabbed UI:** The page now has two tabs: "Basic" and "Advanced".
2.  **Simplified "Basic" Tab:** The "Basic" tab now focuses solely on setting the total monthly spend for each category, removing the individual UPI sliders for a much cleaner look.
3.  **Powerful "Advanced" Tab:** The new "Advanced" tab allows you to set the Card vs. UPI payment split for entire spending *groups* (e.g., Shopping, Bills, Travel) using a single slider for each. This is much faster and more intuitive than managing individual sliders.
4.  **Synchronized Data:** The two tabs are synchronized. Changing a total spend in the "Basic" tab will respect the UPI split you've set in "Advanced," and vice-versa.
